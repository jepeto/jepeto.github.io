## 🎯 Σκοπός

Τα επίσημα ΦΕΚ (Φύλλα Εφημερίδας της Κυβερνήσεως) είναι γραμμένα με νομική γλώσσα που είναι δύσκολο να καταλάβει ο μέσος πολίτης. Αυτό το project μετατρέπει τα επίσημα κείμενα σε απλά άρθρα που μπορεί να καταλάβει ο καθένας.

## 🛠️ Τεχνολογίες

- Vanilla HTML/CSS/JavaScript
- Marked.js για Markdown rendering
- GitHub Markdown CSS για styling
- GitHub Pages για hosting

## 📱 Features

- ✅ Responsive design
- ✅ Dark mode support (follows system)
- ✅ Grouped articles by date
- ✅ Clean, modern UI
- ✅ No build step - pure static files

## 🚀 Local Development

```bash

```
